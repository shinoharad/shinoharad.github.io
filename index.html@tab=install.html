Ниже представлены инструкции по установке и удалению VK Next.
Чётко придерживайтесь их во избежание проблем!

Обратите внимание, что устанавливать обе версии VK Next в один браузер категорически запрещено, это может вызвать критические ошибки. Используйте только одну из версий!

Информация о различии версий находится в разделе Помощь

Установка VK Next
Рекомендуем установить VK Next LiteЭто самостоятельное расширение для ВКонтакте, которое устанавливается без использования Tampermonkey.
Оно содержит почти все функции основного расширения, но при этом гораздо лучше оптимизировано и имеет дополнительные уникальные функции, отсутствующие в script-версии.
Корректная работа данной версии гарантируется только в браузерах на основе Chromium и Firefox с Tampermonkey.

Safari не поддерживается.
  1. Установите расширение Tampermonkey для вашего браузера, VK Next устанавливается через него (при возникновении ошибки используйте VPN)
  2. В браузере с установленным Tampermonkey нажмите кнопку установки скрипта VK Next:
  3. Нажмите кнопку «Установить» в открывшейся вкладке
  4. Перейдите на сайт ВКонтакте (или обновите страницу в уже открытой вкладке VK).
После установки
Обязательно подпишитесь на канал в Telegram или сообщество ВКонтакте. Там оперативно выкладывается важная информация для пользователей расширения!
Остались вопросы? Ответы на многие из них есть в разделе Помощь. Настоятельно рекомендуем ознакомиться!
Удаление VK Next Lite
  1. Нажмите правой кнопкой мыши на иконку VK Next Lite в панели расширений браузера или перейдите в настройки расширений
  2. Нажмите кнопку "Удалить расширение"
  3. Подтвердите удаление в открывшемся окне.
Удаление VK Next
  1. Откройте меню расширения Tampermonkey (клик по иконке в панели расширений браузера)
  2. Выберите "Панель управления"
  3. В открывшейся вкладке нажмите на иконку удаления у скрипта "VK Next", она находится справа
  4. Подтвердите удаление в открывшемся окне.
